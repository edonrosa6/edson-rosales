¿Cómo hacer animaciones en Javascript?

Con javascript podemos hacer distintos tipos de interacciones con el usuario mediante nuestro sitio web como lo son las animaciones, existen librerías como GSAP que hacen animaciones muy fluídas de una manera más fácil y sencilla que te facilitarán el código.

Javascript es un lenguaje de programación interpretado que es muy común en las páginas web, con éste lenguaje podemos tener más interacción con el usuario y hacerlo de una forma dinámica.

A continuación te enseñaré cómo agregar animaciones a nuestra página web con GSAP.

Primero lo que debemos hacer es usar el CDN de GSAP que debemos colocar en la etiqueta <head> de nuestro html.

https://cdnjs.cloudflare.com/ajax/libs/gsap/3.7.1/gsap.min.js

Ahora lo que haremos será poner el CDN del Scroll Trigger que funciona para que al hacer scroll en nuestro sitio web se muestre la animación.

https://cdnjs.cloudflare.com/ajax/libs/gsap/3.7.1/ScrollTrigger.min.js

Ejemplo:

<head>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.7.1/gsap.min.js">
  <script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.7.1/ScrollTrigger.min.js">
</head>

Vamos a ver unos ejemplos de animaciones sencillas con GSAP a continuación:

See the Pen Animacion en GSAP by edonrosa6 (@edonrosa6) on CodePen.

Como verás en el JS aplicamos un gsap.from() para decirle a GSAP que realice la animación desde una posición por ejemplo y: 100 que sería desde abajo, también le pasamos los valores de opacity: 0 para que inicie con una opacidad de cero. La duración deberás colocarla en segundos, es decir, si quieres que la animación dure medio segundo sería duration: .5.

La propiedad ease tiene diferentes valores como :

  • bounce
  • back
  • elastic
  • slow

Si quieres ver los diferentes tipos de ease ve a:

https://greensock.com/docs/v3/Eases

Scroll Trigger

Scroll Trigger se utiliza cuando quieres que un elemento se anime al hacer scroll, también le podemos agregar opciones para que se pause cuando damos scroll muy rápido, entre otras cosas.

Si quieres saber más sobre GSAP ve a su documentación oficial en el siguiente enlace:

https://greensock.com/docs/
